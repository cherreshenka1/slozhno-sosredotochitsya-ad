# Сложно сосредоточиться

Сайт о проблемах концентрации внимания в современном мире. Проект с адаптивной вёрсткой и поддержкой светлой и тёмной тем.

## Описание проекта

«Сложно сосредоточиться» — это информационный сайт, который рассказывает:
- почему современному человеку так трудно концентрироваться
- какие факторы влияют на нашу продуктивность
- что можно сделать, чтобы улучшить фокус внимания

Проект выполнен с использованием методологии **Mobile First** и поддерживает:
- адаптивную вёрстку для мобильных устройств, планшетов и десктопов
- переключение между светлой и тёмной темами (вручную или автоматически)
- декоративные элементы в стиле «неон» и «ретро»

## Технологии

- HTML5
- CSS3
  - Flexbox
  - Grid Layout
  - Position (absolute/relative)
  - Псевдоэлементы (::before, ::after)
  - CSS-переменные (custom properties)
  - Медиазапросы
  - Clamp() для адаптивных шрифтов
- JavaScript (переключение тем — уже реализован)
- Методология БЭМ

## Особенности реализации

### 1. Mobile First подход

Вёрстка начинается с мобильной версии (ширина до 768px), затем добавляются медиазапросы для планшетов (768px+) и десктопов (1024px+).
